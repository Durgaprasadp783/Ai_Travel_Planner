# AI Travel Planner

A full-stack AI-powered travel planning application.

## Project Structure

This project uses a monorepo structure:

- **`/backend`**: Node.js, Express, and MongoDB API.
- **`/frontend`**: Next.js (React) frontend application.

## How to Run

### Backend
```bash
cd backend
npm install
node app.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
